# **Ejercicios de Listas y Tuplas en C#**

Repositorio del proyecto desarrollado para la asignatura Estructura de Datos (E) en la Universidad Estatal Amazónica, carrera de Ingeniería en Tecnologías de la Información.

## **Descripción**

Este proyecto desarrolla una serie de ejercicios prácticos utilizando el lenguaje de programación C#, enfocados en el manejo de listas para almacenar y procesar información. Los ejercicios permiten trabajar con datos como asignaturas, notas, números y precios, mostrando los resultados por pantalla de forma clara y ordenada.

El proyecto aplica conceptos básicos de Programación Orientada a Objetos, organizando cada ejercicio en clases independientes, y fue desarrollado en el entorno Visual Studio Code para su correcta ejecución en consola.

## **Objetivos**

*Aplicar el uso de listas como estructuras de datos dinámicas en C#.
*Reforzar el uso de la Programación Orientada a Objetos mediante clases y métodos.
*Desarrollar programas sencillos que interactúan con el usuario a través de la consola.
*Organizar correctamente un proyecto C# y publicarlo en un repositorio público de GitHub.
